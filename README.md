<h2 style="font-weight:300">How to <strong>run</strong></h2>


After you clone this repository, you need follow the steps below:
-   **With Yarn:**
    1. `yarn`;
    2. `yarn start:dev`
    3. Open `localhost:3000/graphql` and try the queries and mutations.
-   **With NPM:**
    1. `npm install` or `npm i`;
    2. `npm run start:dev`;
    3. Open `localhost:3000/graphql` and try the queries and mutations.
